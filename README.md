# Installation
## Using NPM
Run commands
1) ```cd apps```
2) ```npm install```
3) ```cp .env.example .env```
4) ```npm run start:dev```
## Using Docker
1) ```git clone https://github.com/darkun7/WebAPI-Tokolotik-Tim-Undagi.git```
2) ```docker-compose build```
3) ```docker run -d -p 3000:3000 [image id]```
